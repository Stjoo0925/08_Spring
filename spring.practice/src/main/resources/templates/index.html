3가지 컨트롤러를 만든다.
url은 자유로 하며 index에서는 모든 페이지를 이동할 수 있도록 구성한다.

1. index를 연결하는 컨트롤러

2. 사용자가 입력한 쿼리 스트링 파라미터 값을 h1태그를 이용하여 표시하는 컨트롤러

3. 사용자의 권한을 select box(2가지)로 선택하여 1개는 정상응답 1개는 비정상 응답으로 구분
3-1 정상응답 : 사용자가 입력한 메세지를 담아서 화면에 표시
3-2: 비정상응답 : 사용자에게 잘못된 요청이라는 값을 담아서 index에서 alert로 표시
